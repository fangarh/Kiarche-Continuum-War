# Рабочий контекст Antigravity (Kiarche Continuum War)

Дата создания: 2026-04-02

## 1. Обзор репозитория
Проект разделен на две основные части:
1. **Прототип игры** `KiarcheContinuumWar/` (Unity 6.0 / 6000.4.0f1 URP).
2. **Документационный портал** `documentation/kcwweb/` (React 19, TypeScript 5.9, Vite 8).

Также имеется база знаний и лора в `documentation/memory/`.

## 2. Ключевые правила
- **Язык общения:** ВСЕГДА отвечать пользователю на русском языке (`memory/user_language.md`).
- **Unity Проект:**
  - Текущий статус: MVP RTS с базовым выделением и перемещением (реализовано 2026-03-28).
  - Скрипты управления: `RTSInput.cs`, `UnitController.cs`, `Unit.cs`.
  - Требует доработки: Индикатор атаки, звуки, система боя (урон/хп), ресурсы.
- **Web Проект (kcwweb):**
  - Data-Driven Architecture: Весь текстовый контент должен находиться в конфигурациях TypeScript в `src/data/`, а не хардкодиться в React-компонентах.
  - Каждая страница имеет собственный CSS-файл.
  - Запуск: `npm install` && `npm run dev` в директории `documentation/kcwweb/`.
- **Лор вселенной:**
  - Источник истины: `documentation/memory/stargates_lore_discussion.md`.
  - Играбельные фракции: Молодые расы (Ре'Зиры, Ти'Оны, Ве'Ори, Родверы) и Тал'Син (механика идентичности).
  - Названия фракций изменены от оригинальных референсов: Architects -> Синтекс, Goa'улды -> Вел'Кеты, Асгарды -> Кешари, Ноксы -> Сил'Ни, Древние -> Этерны.

## 3. Мои цели как разработчика-ассистента
Перед началом изменений в коде я буду сверяться с:
- Статусом RTS MVP (`documentation/memory/rts_mvp_progress.md`).
- Наличием существующих компонентов в `kcwweb` (использовать `src/components/ui/`).
- Положениями лора.
