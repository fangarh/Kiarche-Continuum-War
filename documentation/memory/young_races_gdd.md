## УТВЕРЖДЁННЫЕ КОНЦЕПЦИИ

### 1. Ре'Зиры (Re'Zir) — Последователи Вел'Кетов

**Прообраз:** Джунари

#### Биология
- Гуманоиды, генетически модифицированные Вел'Кетами для жизни в экстремальных условиях
- Встроенный "нанитовый симбионт", дающий усиление и долголетие
- Утратили оригинальную биологию, адаптированы под рабство

#### Культура
- **Культ Вел'Кетов как богов** — искренняя вера, не принуждение
- **Иерархия через близость к богу** — чем ближе к Вел'Кету, тем выше статус
- **Потеря идентичности** — нет личных имён, только титулы

#### Базовые механики
- **Нанитовый симбионт** — усиление юнитов, но без свободы воли
- **Орды** — дешёвые юниты с бонусом к количеству
- **Захват** — бонус к захвату городов, штраф к технологиям
- **Божественный приказ** — Вел'Кеты могут управлять юнитами Ре'Зиров

#### Технологии
```
Клинки → Нанитовые усилители → Энергетическое оружие → Космические перелёты
```

#### Уникальная фишка
**Коллективная воля** — бонус к координации, но не могут иметь героев (только Вел'Кет-надзиратель).

### 2. Ти'Оны (Ti'On) — Последователи Кешари

**Прообраз:** Берсеркеры (прямое влияние Asgard)

#### Биология
- Гуманоиды, генетически стабилизированные Кешари
- Встроенный "мозговой имплант" — нейроинтерфейс для телепатии
- Стройные, серебристые, со светящимися глазами
- Долгая жизнь, зависят от Кешари-технологий для регенерации

#### Культура
- **Протекторат как родина** — неотделимы от Кешари, это "семья"
- **Технологический орден** — каждый — инженер или учёный
- **Телепатическая демократия** — решения через консенсус разума
- **Виртуальное бессмертие** — сознание сохраняется в сети

#### Базовые механики
- **Клонирование** — восстановление армии
- **Телепатия** — бонус к дипломатии и разведке
- **Технологический бонус** — стартовые технологии выше
- **Зависимость** — без Кешари-технологий деградируют

#### Технологии
```
Сканеры → Нейросети → Клонирование → Квантовые компьютеры
```

#### Уникальная фишка
**Коллективный разум** — объединение юнитов для усиления, но теряется индивидуальность.

### 3. Ве'Ори (Ve'Ori) — Последователи Этернов

**Прообраз:** Фурлинги (следующие учению о Вознесении)

#### Биология
- Гуманоиды, способные впадать в "созерцательное состояние"
- Во время созерцания — контакт с "Этернальным полем"
- Генетическая связь с Этернами
- Кожа с меняющимися рисунками при "пробуждении"

#### Культура
- **Учение о Вознесении** — цель: подготовиться к Вознесению
- **Ненасилие как догма** — не убивают, даже врагов
- **Хранители знаний** — смерть = передача знаний
- **Иерархия мудрости** — чем ближе к Вознесению, тем выше статус

#### Базовые механики
- **Наследие Этернов** — бонус к исследованиям
- **Созерцание** — юнит получает временный бонус
- **Передача знаний** — погибшие передают опыт без потери
- **Антагонизм** — Вел'Кеты их ненавидят

#### Технологии
```
Религиозные тексты → Символы-ключи → Энергетические маяки → Пустота
```

#### Уникальная фишка
**Пробуждение** — критический момент даёт сильный бонус, но после — временная слабость.

### 4. Родверы (Rodver) — Самостоятельный путь

**Прообраз:** Люди СГ-1 — независимая цивилизация

#### Биология
- Генетически модифицированные люди, минимальные изменения
- **Усиленная адаптивность** — способны быстро приспосабливаться к любой среде
- **Резистентность к радиации** — выживаемость в экстремальных условиях
- Без внешних модификаций — компенсируют технологиями

#### Культура
- **Независимость** — ни от Вел'Кетов, ни от Кешари, ни от Этернов
- **Прагматизм** — результат важнее метода
- **Коллективизм** — "род" как основа (семья, община)
- **Технологический прогресс** — интенсивно развивают технологии

#### Базовые механики
- **Адаптивность** — бонус при закреплении на любом типе карты и ресурсной точки
- **Технологии** — ускоренное развитие индустрии
- **Родовые связи** — бонус к производству через " общину"
- **Универсальность** — нет явных слабостей

#### Технологии
```
Инструменты → Машины → Заводы → Автоматизация → ИИ
```

#### Уникальная фишка
**Адаптивное производство** — здания автоматически модифицируются под текущую потребность (изменение режима работы).
