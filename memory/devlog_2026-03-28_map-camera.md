### ✅ RTS Камера (task-019)

**Статус:** Завершено  
**Время реализации:** ~1.5 часа  
**Файлы:** 1 скрипт + документация

#### RTSCamera.cs

**Функционал:**
- WSAD панорамирование
- Зум колёсиком мыши (10-50 единиц)
- Ограничение границами карты
- Плавное движение (lerp, smoothSpeed)
- Follow за целью (SetTarget)
- Телепорт (Teleport)
- Перемещение к спавну (MoveToPlayerSpawn)

**Управление:**

| Клавиша | Действие |
|---------|----------|
| W / ↑ | Движение вперёд |
| S / ↓ | Движение назад |
| A / ← | Движение влево |
| D / → | Движение вправо |
| Колёсико | Зум (10-50 единиц) |

**Параметры:**
```
Camera Settings:
  Move Speed: 20
  Zoom Speed: 10
  Min Zoom: 10
  Max Zoom: 50
  Default Zoom: 30

Rotation Settings:
  Default Rotation X: 45    # Изометрический вид

Bounds:
  Limit To Map Bounds: true
  Map Bounds: (100, 100)
```

## 🔧 Интеграция

### Настройка сцены

1. **Создать карту:**
   ```
   Tools → KCW → Generate Test Map
   Save: TestMap.unity
   ```

2. **Добавить камеру:**
   - Main Camera → Add Component → RTSCamera
   - Настроить: zoom=30, speed=20

3. **Настроить FlowField:**
   - FlowFieldManager: Field Width/Height = 100
   - Cell Size = 1

4. **Запустить:**
   - WSAD: перемещение камеры
   - Колёсико: зум
   - ПКМ: приказ юнитам
   - Юниты обходят препятствия

### Архитектура

```
MapManager (границы 100x100)
    ↓
RTSCamera (ограничение границами)
    ↓
FlowFieldManager (поле 100x100)
    ↓
Obstacle (регистрация препятствий)
    ↓
UnitPathfinder (движение + обход)
```
