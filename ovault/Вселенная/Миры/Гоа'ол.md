# Гоа'ол

> [!danger] Вулканическая цитадель Вел'Кетов, родной мир паразитов

Гоа'ол — вулканическая планета, родной мир Вел'Кетов и центр их империи. Здесь, в экстремальных условиях подземных полостей и лавовых полей, зародилась паразитическая раса и отсюда начала свою экспансию.

## Физические характеристики

| Параметр | Значение |
|----------|----------|
| **Тип** | Вулканическая планета |
| **Атмосфера** | Плотная, горячая, насыщенная серой и металлическими соединениями |
| **Гравитация** | 1.2g (повышенная) |
| **Температура поверхности** | 200-800°C |
| **Температура подземных полостей** | 40-120°C (обитаемые зоны) |
| **Спутники** | 2 — Кет'Ра (красный), Кет'Ни (серый) |

## География

### Поверхность

Поверхность Гоа'ола — адский пейзаж:
- **Лавовые моря** — огромные озёра расплавленной магмы, занимающие 60% поверхности
- **Вулканические кальдеры** — гигантские кратеры, некоторые диаметром в сотни километров
- **Пепловые пустыни** — равнины, покрытые вулканическим пеплом и металлическими отложениями
- **Горы из застывшей лавы** — острые, чёрные хребты, пронзающие атмосферу

Поверхность необитаема для большинства форм жизни. Только Вел'Кеты в защищённых носителях могут выживать здесь короткое время.

### Подземные полости

Жизнь Гоа'ола сосредоточена **под поверхностью**:
- **Пещерные системы** — огромные подземные полости, где температура и давление позволяют существовать жизни
- **Подземные океаны** — резервуары горячей воды, обогащённой минералами
- **Термальные источники** — источники энергии для подземных экосистем
- **Биолюминесцентные зоны** — области, освещённые светящимися организмами

### Руины Синтекс

В глубочайших полостях Гоа'ола находятся руины станции Синтекс — заброшенного комплекса, который изменил судьбу До-Вел'Кетов:
- **Центральный зал** — огромный зал с субпространственными генераторами (неактивны)
- **Хранилище нанитов** — источник нанитов, мутировавших До-Вел'Кетов
- **Архив** — повреждённые данные о Portal Network (частично расшифрованы Вел'Кетами)
- **Инкубатор-прототип** — первый инкубатор, построенный Вел'Кетами на основе технологий Синтекс

## Ключевые локации

| Локация | Описание |

| **Инкубаторы Гоа'ола** | Крупнейшие производственные комплексы — 10000+ личинок в месяц |
| **Космопорт** | Орбитальный лифт и посадочные площадки для флота Вел'Кетов |
| **Зал Владыков** | Место собраний Владыков — каждый имеет трон, но трон Ра Кена возвышается над остальными |
| **Тюрьма сознаний** | Подземная камера, где хранятся подавленные сознания особо опасных носителей |
| **Полигон** | Зона боевых тренировок — Ре'Зиры и Чистые Воины отрабатывают тактику |

| **Ре'Зиры** | ~2 000 000 | Коренное население, рабочая сила и армия |
| **Порабощённые** | ~500 000 | Представители других рас, привезённые на Гоа'ол |

## Экономика

Гоа'ол — не производственный мир. Его экономика основана на:
- **Контроль** — управление империей Вел'Кетов из цитадели
- **Производство личинок** — крупнейшие инкубаторы снабжают всю империю
- **Исследования** — Жрецы разрабатывают новые технологии паразитирования
- **Военное командование** — координация всех армий Вел'Кетов

Ресурсы поступают с порабощённых миров. Гоа'ол потребляет, не производя (кроме личинок).

## Оборона

Гоа'ол — самая защищённая планета империи:
- **Орбитальный щит** — энергетический барьер, защищающий планету от бомбардировки
- **Орбитальные платформы** — станции с плазменными пушками
- **Флот обороны** — постоянная группировка из 50+ Хексов
- **Наземная оборона** — Чистые Воины, элитные носители, нанитовые ловушки
- **Подземные бункеры** — Вел'Кеты могут пережить любую бомбардировку в глубинах планеты

## Значение

Гоа'ол — сердце империи Вел'Кетов. Потеря Гоа'ола = крах империи. Ра Кен знает это и укрепляет планету тысячелетиями.

Для других фракций:
- **Кешари** — хотят уничтожить инкубаторы, но не могут пробить оборону
- **Ре'Зиры** — считают Гоа'ол священным местом, «домом богов»
- **Апофис** — мечтает захватить Гоа'ол и доказать своё превосходство
