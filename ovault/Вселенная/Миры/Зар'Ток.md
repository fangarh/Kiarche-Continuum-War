# Зар'Ток

> [!danger] Фронтирный мир Вел'Кетов, передовая экспансии

Зар'Ток — недавно захваченная планета на границе пространства Вел'Кетов. Стратегический форпост для дальнейшей экспансии через Portal Network. Место постоянного конфликта с силами Кешари и молодыми расами.

## Физические характеристики

| Параметр | Значение |
|----------|----------|
| **Тип** | Умеренная планета, ранее обитаемая |
| **Атмосфера** | Пригодна для дыхания (частично загрязнена) |
| **Гравитация** | 1.0g |
| **Температура** | -10 до 35°C (сезонные колебания) |
| **Спутники** | 3 |

## История

Зар'Ток был миром независимой гуманоидной расы с технологическим уровнем, близким к индустриальной эпохе. Вел'Кеты захватили его 50 лет назад через Portal Network, используя классическую тактику: дестабилизация → паразитирование лидеров → полный захват.

**Текущий статус:** Планета частично контролируется Вел'Кетами. Значительная часть поверхности — зона сопротивления. Ре'Зиры-гарнизон ведёт постоянную войну с партизанами.

## Ключевые локации

| Локация | Описание |
|---------|----------|
| **Портал** | Ключевой объект — врата Синтекс, через которые Вел'Кеты вошли. Укреплён и охраняется |
| **Цитадель оккупации** | Военная база Вел'Кетов — командный центр, инкубатор, тюрьма |
| **Зона сопротивления** | Горные регионы, где укрываются партизаны. Неполный контроль Вел'Кетов |
| **Разрушенные города** | Бывшие столицы цивилизации — сейчас руины, символ сопротивления |
| **Лагеря Ре'Зиров** | Военные поселения Ре'Зиров, гарнизон планеты |

## Конфликт

Зар'Ток — арена постоянного конфликта:
- **Вел'Кеты** контролируют города, портал и ключевую инфраструктуру
- **Сопротивление** контролирует горные регионы и ведёт партизанскую войну
- **Кешари** тайно снабжают сопротивление оружием и технологиями
- **Родверы** имеют разведывательную станцию на одном из спутников

## Значение

Зар'Ток — пример того, как Вел'Кеты ведут экспансию, и пример того, что их можно остановить. Планета демонстрирует:
- Уязвимость Вел'Кетов на фронтире
- Способность молодых рас к сопротивлению
- Важность Portal Network как стратегического объекта

Для Апофиса Зар'Ток — возможность доказать эффективность своей военной доктрины. Для Ра Кена — тест на то, может ли его система контроля удержать далёкий мир.
