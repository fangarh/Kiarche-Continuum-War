# Отказ Ки'Архе

> [!abstract] Эра Архитекторов — Древность

Ки'Архе отвергают Portal Network по семи причинам:

1. **Разрыв непрерывности** — дискретные узлы вместо непрерывной структуры
2. **Централизация и уязвимость** — критические точки отказа
3. **Энтропийный паразитизм** — разрыв связи между затратами и последствиями
4. **Искажение эволюции** — искусственно равные условия для неравных уровней
5. **Потеря контроля над идентичностью** — копирование, подмена, модификация
6. **Протокольная зависимость** — необходимость доверия к протоколу
7. **Ложная метрика развития** — иллюзия прогресса через скорость экспансии
