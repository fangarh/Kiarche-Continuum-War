# Создание Portal Network

> [!abstract] Эра Архитекторов — Древность

Синтекс создают сеть порталов, связывающих тысячи миров. Технология основана на стабилизации разрывов в пространстве — «складках», позволяющих мгновенное перемещение. Ки'Архе отказываются участвовать в проекте, считая его концептуальной ошибкой.

## Цели создания

1. **Rift** — быстрое перемещение ресурсов для создания линий обороны
2. **Изоляция цивилизаций** — сеть обеспечивала обмен знаниями
3. **Энтропия** — инфраструктура как способ противостоять распаду
