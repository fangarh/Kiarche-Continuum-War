# Экономическая система

> [!abstract] Ресурсы, валюты и торговля

Экономика строится на ресурсах, торговле и контроле над Portal Network. Для Тал'Син ключевой ресурс — Знание.

## Валюты Тал'Син

| Валюта | Символ | Описание |
|--------|--------|----------|
| Резонанс | ⟁ | Основная валюта |
| Знание | Ξ | Из интеграции погибших юнитов |
| Непрерывность | ∮ | Редкий ресурс |

## Ресурсы

| Ресурс | Тип | Описание |
|--------|-----|----------|
| Резонансные якоря | Обычный | Точки для мгновенного перемещения |
| Кристаллические осколки | Обычный | Остатки материала Тал'Син |
| Фрагменты Эхо | Редкий | Нестабильные остатки интеграции |
| Ядро непрерывности | Легендарный | Запись полной идентичности |
