# Portal Network

> [!abstract] Сеть древних порталов, связывающая планеты

Сеть порталов, созданная Синтекс (Architects), связывающая тысячи миров. Технология основана на стабилизации разрывов в пространстве — «складках», позволяющих мгновенное перемещение.

## Механика

- **Генерация сети** — процедурное размещение порталов
- **Активация порталов** — стоимость и условия
- **Телепортация** — перемещение между порталами
- **Колонизация** — основание новых колоний через порталы
- **Захват порталов** — военные действия за контроль
- **Особые порталы** — легендарные локации

## Альтернатива: Резонансная модель Ки'Архе

Ки'Архе не используют Portal Network. Вместо этого — резонансные якоря:
- Нет фиксированных узлов
- Перенос состояния сохраняет непрерывность
- Взаимодействие только при структурной совместимости
