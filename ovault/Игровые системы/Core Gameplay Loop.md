# Core Gameplay Loop

> [!abstract] eXplore, eXpand, eXploit, eXterminate

Основной цикл игры — классическая 4X-механика. Герои служат инструментом для реализации каждого из этапов.

## Шаги цикла

| # | Этап | Описание | Для Тал'Син |
|---|------|----------|-------------|
| 1 | **Исследование** (Explore) | Разведка новых миров через порталы | Поиск резонансных якорей |
| 2 | **Экспансия** (Expand) | Основание колоний | Создание узлов синхронизации |
| 3 | **Эксплуатация** (Exploit) | Развитие ресурсов и технологий | Накопление знаний и структуры |
| 4 | **Уничтожение** (Exterminate) | Сражения с врагами | Синхронизация героев усиливает армию |
