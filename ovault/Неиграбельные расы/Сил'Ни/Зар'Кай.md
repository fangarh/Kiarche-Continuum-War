# Зар'Кай

> [!abstract] Родная планета Фен'Ри — живой организм, не просто мир

Зар'Кай — планета, где Фен'Ри развились естественным путём. Это не просто мир — это **живой организм**, где цивилизация и природа едины.

---

## Обзор

Зар'Кай — зелёно-голубая планета с умеренным климатом. Поверхность покрыта густыми лесами, кристальными реками и живыми поселениями Фен'Ри.

**Ключевая особенность:** Фен'Ри не доминируют над природой — они **часть** экосистемы. Их поселения выращены, не построены. Их технологии — биоорганические, не механические.

---

## Живая архитектура

| Структура | Описание |
|-----------|----------|
| **Деревья-башни** | Выращенные жилища, растущие вверх. Каждый вид Фен'Ри имеет свой тип дерева |
| **Мосты-лианы** | Живые переходы между строениями. Растут и адаптируются к нагрузке |
| **Реки-нервы** | Водные пути, несущие телепатические сигналы. Связывают поселения в единую сеть |
| **Живые щиты** | Симбиотические организмы, создающие энергетические барьеры вокруг поселений |
| **Циркл-Поляна** | Место совета старейшин всех видов. Живой амфитеатр, выращенный из единого дерева |

## Экосистема

Фен'Ри осознанно поддерживают баланс:

| Роль | Вид | Функция |
|------|-----|---------|
| **Хищники** | Волки, лисы | Контроль популяций |
| **Травоядные** | Олени, копытные | Удобрение земли |
| **Наблюдатели** | Птицы | Разнос семян, разведка |
| **Инженеры** | Грызуны, мелкие | Переработка, целительство |
| **Защитники** | Рептилии | Охрана границ |
