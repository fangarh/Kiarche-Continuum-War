# Старейшие личинки

> [!danger] Редкая стадия развития Вел'Кетов

## Описание

**Старейшая личинка** — личинка Вел'Кетов, не нашедшая носителя за 50+ лет. Развивает **собственное сознание**, независимое от колонии.

## Свойства

| Свойство | Описание |
|----------|----------|
| **Независимость** | Не подчиняется Владыкам |
| **Избирательность** | Паразитирует выборочно, не массово |
| **Собственное сознание** | Развивает уникальную личность |

## Типы Старейших

| Тип | Описание | Роль в игре |
|-----|----------|-------------|
| **Отшельник** | Живёт в глубинах Гоа'ола, избегает общества | NPC-источник знаний |
| **Еретик** | Утверждает: паразитизм — ошибка, нужно вернуться к симбиозу | Квестодатель, потенциальный союзник |
| **Мудрец** | Накопил знания за десятилетия без носителя | Торговец информацией |

## В игре

Старейшие личинки — **уникальные NPC**:
- Источники знаний о слабостях Вел'Кетов
- Квестодатели для пути Бунтарей Ре'Зиров
- Потенциальные союзники (если игрок идёт путём освобождения)

### Способности Старейшего

| Способность | Описание |
|-------------|----------|
| **«Древнее знание»** | Раскрывает слабые места Вел'Кетов |
| **«Независимая воля»** | Иммунитет к контролю Владык |
| **«Симбиоз»** | Может усилить носителя без подавления (возврат к корням) |
