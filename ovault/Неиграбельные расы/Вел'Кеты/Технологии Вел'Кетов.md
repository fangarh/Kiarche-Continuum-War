# Технологии Вел'Кетов

> [!danger] Адаптированные, не изобретённые

Вел'Кеты не изобретают технологии — они **захватывают, адаптируют и искажают**. Их технологическая база построена на трёх столпах: руины Синтекс, технологии порабощённых рас и собственные биологические разработки.

## Паразитические системы

### Личинки

**Описание:** Основная форма Вел'Кета. Сегментированное тело размером с ладонь, золотисто-багрового цвета. Нервные отростки на конце тела используются для подключения к стволу мозга носителя.

**Производство:**
- **Биологическое** — зрелый Вел'Кет производит личинок через внутренний процесс, аналогичный делению клеток. 1 Вел'Кет = 1-3 личинки в год
- **Инкубаторное** — биомеханические комплексы массового производства. Один инкубатор производит до 1000 личинок в месяц
- **Нанитовое клонирование** — экспериментальная технология, создающая личинок из чистой нанитовой материи. Нестабильно, но не требует биологического Вел'Кета

**Стадии развития личинки:**
| Стадия | Длительность | Описание |
|--------|-------------|----------|
| **Яйцо** | 2-4 недели | Формирование в инкубаторе или теле Вел'Кета |
| **Молодая** | 1-2 недели | Активный поиск носителя. Может выживать вне тела до 48 часов |
| **Зрелая** | 10-50 лет | Полноценный паразит. Способна производить нанопаразитов |
| **Старейшая** | 50+ лет | Редкая стадия. Личинка, не нашедшая носителя, развивает собственное сознание |

### Нанопаразиты

**Описание:** Микроскопические организмы, производные Вел'Кетов. Размер — несколько микрон. Распространяются через кровоток носителя, формируя паразитическую сеть.

**Функции:**

| Функция | Описание |
|---------|----------|
| **Контроль** | Блокируют моторные нейроны носителя, передавая команды Вел'Кета. Сознание носителя остаётся активным, но не может управлять телом |
| **Усиление** | Выделяют стимуляторы, повышающие силу, скорость и регенерацию носителя. Побочный эффект — сокращение жизни на 10-30% |
| **Связь** | Создают нанитовую сеть между всеми носителями одного Владыки. Позволяет координировать действия на расстоянии |
| **Наказание** | При команде Вел'Кета могут вызвать боль, паралич или смерть носителя. Используется для подавления бунтов |
| **Мониторинг** | Передают Вел'Кету сенсорную информацию от носителя — зрение, слух, тактильные ощущения |

### Инкубаторы

**Описание:** Биомеханические комплексы для массового производства личинок. Сочетают органические ткани Вел'Кетов с технологиями Синтекс.

**Структура:**
- **Биокамеры** — органические резервуары, где выращиваются личинки
- **Нанитовые инжекторы** — вводят наниты в развивающиеся личинки для усиления
- **Контрольный узел** — Вел'Кет-Жрец управляет процессом через нанитовую связь

**Производственные мощности:**
- Малый инкубатор: 100-500 личинок/месяц
- Средний инкубатор: 1000-5000 личинок/месяц
- Большой инкубатор (Гоа'ол): 10000+ личинок/месяц

## Вооружение

### Жезлы Вел'Кетов

**Описание:** Основное энергетическое оружие. Стреляет концентрированными плазменными зарядами, способными пробивать броню и энергетические щиты.

**Принцип действия:** Жезл генерирует плазменный заряд через нанитовый реактор в рукояти. Заряд формируется в форме змеиной головы — не для красоты, а для фокусировки энергии.

**Варианты:**
| Тип | Мощность | Описание |
|-----|----------|----------|
| **Стандартный** | Средняя | Оружие рядовых носителей. Пробивает стандартную броню |
| **Усиленный** | Высокая | Оружие командиров. Пробивает щиты Кешари |
| **Жезл Владыки** | Экстремальная | Персональное оружие Владык. Залп уничтожает бронетехнику |
| **Парные** | Переменная | Оружие Апофиса. Один для атаки, другой для щита |

### Кольца телепортации

**Описание:** Устройства мгновенного перемещения, скопированные с технологии Синтекс. Два кольца создают связь — шаг через одно = появление через другое.

**Ограничения:**
- Требуют пары — нельзя телепортироваться в случайное место
- Ограниченная дальность — в пределах одной планетарной системы
- Уязвимы к помехам — Кешари разработали устройства блокировки

**Использование:**
- Быстрая переброска войск
- Эвакуация Владыков
- Диверсионные операции

### Щиты Вел'Кетов

**Описание:** Энергетические барьеры, защищающие от физического и энергетического урона.

**Типы:**
| Тип | Покрытие | Описание |
|-----|----------|----------|
| **Персональный** | Один носитель | Нанитовый щит, генерируемый нанопаразитами в теле |
| **Тактический** | Группа до 50 юнитов | Портативный генератор щита, носимый Ре'Зирами |
| **Стационарный** | Здание/база | Мощный генератор, защищающий ключевые объекты |
| **Орбитальный** | Корабль/планета | Щит материнского корабля или цитадели |

### Дополнительное вооружение

| Оружие | Описание |
|--------|----------|
| **Плазменные пушки** | Тяжёлое вооружение кораблей. Разрушают здания и укрепления |
| **Нанитовые бомбы** | Оружие массового поражения. Распространяют нанопаразитов в атмосфере, подчиняя всё живое |
| **Усилители подавления** | Устройства, усиливающие контроль Вел'Кета над носителями в радиусе. Используются для подавления бунтов |
| **Нейро-ловушки** | Устройства для захвата разумных существ alive — парализуют нервную систему, подготавливая к паразитированию |

## Корабли

### Материнские корабли

**Описание:** Гигантские орбитальные крепости — флагманы флота Вел'Кетов. Длина до 5 км.

**Функции:**
- Командный центр Владыки
- Производство личинок (мобильный инкубатор)
- Производство боевых кораблей
- Орбитальная бомбардировка

**Вооружение:**
- Плазменные батареи (основное)
- Нанитовые бомбардировщики
- Щит орбитального уровня
- Кольца телепортации для десанта

### Хексы

**Описание:** Стандартные боевые корабли Вел'Кетов. Шестиугольная форма (отсюда название). Длина 200-500 м.

**Характеристики:**
- Экипаж: 50-200 носителей
- Вооружение: плазменные пушки, торпеды
- Щит: тактический уровень
- Десантный отсек: до 500 Ре'Зиров

**Роль:** Основная боевая единица флота. Действуют группами по 6-12 кораблей.

### Десантные корабли

**Описание:** Транспортные корабли для переброски войск на поверхность.

**Характеристики:**
- Вместимость: 1000-5000 юнитов
- Вооружение: минимальное (только самооборона)
- Скорость: высокая
- Щит: базовый

### Разведывательные зонды

**Описание:** Маленькие беспилотные аппараты для разведки. Маскируются под астероиды или космический мусор.

**Функции:**
- Сканирование планет
- Перехват коммуникаций
- Обнаружение порталов
- Передача данных через нанитовую сеть

## Технологии контроля

### Нанитовые сети

**Описание:** Глобальная система контроля, связывающая всех носителей одного Владыки через нанопаразитов.

**Возможности:**
- Координация армий в реальном времени
- Мгновенная передача приказов
- Мониторинг лояльности
- Удалённое наказание (боль/смерть)

**Слабости:**
- Кешари разработали устройства блокировки нанитовой связи
- Слишком плотная сеть создаёт единую точку отказа
- Ре'Зиры с усиленными имплантами могут частично сопротивляться

### Усилители подавления

**Описание:** Устройства, усиливающие контроль Вел'Кета над носителем. Используются для:
- Подавления сильных личностей (лидеров, героев)
- Усиления контроля над старыми носителями
- Подавления бунтов

**Принцип действия:** Устройство генерирует субпространственное поле, усиливающее сигнал нанопаразитов. Носитель теряет даже минимальную автономию.

### Нейро-интерфейсы

**Описание:** Устройства для прямого подключения к нервной системе носителя. Позволяют:
- Извлекать воспоминания и знания
- Модифицировать личность (стирать, добавлять)
- Создавать «идеальных» носителей с заданными навыками

**Этический статус:** Даже по меркам Вел'Кетов это считается крайней мерой. Использование нейро-интерфейсов разрушает личность носителя, делая его пустой оболочкой.

## Технологическое дерево

```
Этап 1: Базовые технологии
├── Личинки (биологическое производство)
├── Жезлы (стандартные)
├── Кольца телепортации (копия Синтекс)
└── Базовые щиты

Этап 2: Расширение
├── Инкубаторы (массовое производство)
├── Нанопаразиты (контроль на расстоянии)
├── Хексы (боевые корабли)
└── Нанитовые сети (координация)

Этап 3: Превосходство
├── Нанитовое клонирование личинок
├── Жезлы Владыки (усиленные)
├── Материнские корабли
├── Нанитовые бомбы
└── Нейро-интерфейсы

Этап 4: Абсолютный контроль
├── Глобальная нанитовая сеть
├── Усилители подавления
├── Орбитальные щиты
└── Мобильные инкубаторы
```
