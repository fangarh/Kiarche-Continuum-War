# Промпт для генерации концепт-арта: Ка'Рра

Midjourney v6 / DALL-E 3 / Stable Diffusion

## Контекст вселенной

**Kiarche Continuum War** — научно-фантастическая RTS-стратегия. Фен'Ри — антропоморфные разумные звери планеты Зар'Кай, последователи пацифистской расы Сил'Ни.

**Ка'Рра** — пантера-защитник. Тихий защитник — скрытность + мощь. Сила без агрессии. Не хищный супергерой, а древний лесной страж Зар'Кай: сдержанный, тяжёлый, терпеливый, связанный с живой архитектурой мира.

Атмосфера должна передавать древнюю бдительность, тихий авторитет, сдержанность и философию «сила без агрессии». Цифровая живопись, приземлённый science-fantasy концепт-арт, портрет персонажа.
